# ASTU Computer Science Department Lab Building (FreeGLUT)

A 3D graphics model of the **Adama Science and Technology University (ASTU) Computer Science Department Lab Building** developed using **C++ and FreeGLUT/OpenGL**.

## Features
- 3D visualization of the lab building
- Basic geometric modeling
- Camera navigation
- OpenGL rendering with FreeGLUT

## Technologies Used
- C++
- OpenGL
- FreeGLUT

## Purpose
This project was developed for computer graphics practice and demonstrates 3D modeling concepts using FreeGLUT.

## How to Run
1. Install FreeGLUT and OpenGL libraries
2. Compile the source code
3. Run the executable
